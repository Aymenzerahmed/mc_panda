mc-panda
=======

This package provides a RobotModule implementation for the following robots:
- panda and variants

Dependencies
------------

This package requires:
- [mc_rtc]
- [panda_cnoid]

[panda_cnoid]: https://github.com/gergondet/panda_cnoid
[mc_rtc]: https://gite.lirmm.fr/multi-contact/mc_rtc
